Погодный Бот Telegram 
Этот бот для Telegram предоставляет информацию о текущей погоде в любом городе мира.
Просто введите название города, и бот отправит вам:

Текущую температуру в градусах Цельсия.
Картинку, соответствующую погоде (солнечно или пасмурно).


Основные возможности:
Получение погоды по названию города.
Отображение температуры в градусах Цельсия.
Автоматическая отправка изображения, отражающего погодные условия.
Уведомление в случае неправильного ввода или отсутствия данных для указанного города.
Используемые технологии:


Telegram Bot API (через библиотеку pyTelegramBotAPI)
OpenWeatherMap API (для получения данных о погоде)
